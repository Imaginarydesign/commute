# Commute

Discover the cost of your commute

![todo](https://github.com/Imaginarydesign/commute/raw/master/image.png)

## Installation

```sh
$ git clone https://github.com/Imaginarydesign/commute.git
$ cd commute
$ bower install
```
To run locally:

```sh
$ gulp serve
```
